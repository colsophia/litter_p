# 七：MySQL存储过程

教程 (opens new window)

下面为个人简练了教程总结出来的

# 1、存储过程的创建和调用

创建存储过程：

DELIMITER & -- 自定义语句的结束符&
CREATE PROCEDURE my_proceduer() -- CREATE PROCEDURE 存储过程名(参数)
BEGIN  -- 存储过程开始

end&  -- 存储过程结束

# 2、存储过程的参数

三种类型：IN、OUT、INOUT

  • IN：输入参数，表示调用者向存储过程传入值【字面量、变量】
  • OUT：输出参数，表示存储过程向调用者传出值【变量】
  • INOUT：输入输出参数，表示调用者和存储过程两者间可以相互传出传入值【变量】
