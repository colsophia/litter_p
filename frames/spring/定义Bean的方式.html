# 四：定义Bean的方式

# 1、JavaBean、SpringBean、对象之间的区别

Bean肯定是对象

# 1.1 什么是JavaBean:

  1、是一个Class类
  2、所有的属性都私有化（private）
  3、必须具有无参数的构造器
  4、通过提供setter和getter方法来实现对成员属性的访问

public class UserInfo {
    private String userName;

    public String getUserName() {
        return userName;
    }

    public void setUserName(String userName) {
        this.userName = userName;
    }
}

# 1.2 什么是SpringBean

凡是由Spring生成的对象就称为Spring的Bean

// 通过bean配置，获取userInfo SpringBean对象
ClassPathXmlApplicationContext applicationContext = new ClassPathXmlApplicationContext("spring.xml");
UserInfo userInfo1 = applicationContext.getBean("userInfo", UserInfo.class);

# 2、定义Bean的方式

声明式：xml、@Bean、@Component
编程式：BeanDefinition
其中声明式编程的底层就是BeanDefinition
FactoryBean方式、Supplier方式

# 2.1 xml方式

在resource文件夹下创建一个管理bean的xml文件

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">

    <bean id="userInfo" class="com.example.springboot.domain.UserInfo"/>
</beans>

读取xml文件，获取bean对象

ClassPathXmlApplicationContext applicationContext = new ClassPathXmlApplicationContext("spring.xml");
UserInfo userInfo = applicationContext.getBean("userInfo", UserInfo.class);

# 2.2 @Bean方式


public class SpringConfig {

    @Bean
    public UserInfo userInfo() {
        return new UserInfo();
    }
}

AnnotationConfigApplicationContext applicationContext = new AnnotationConfigApplicationContext(SpringConfig.class);
UserInfo userInfo = applicationContext.getBean("userInfo", UserInfo.class);

# 2.3 @Component方式

//扫描com.example.springboot路径下的@Component注解，扫描到了就会认为是一个SpringBean
@ComponentScan("com.example.springboot")
public class SpringConfig {
}

@Component
public class UserInfo {
    private String userName;

    public String getUserName() {
        return userName;
    }

    public void setUserName(String userName) {
        this.userName = userName;
    }
}

AnnotationConfigApplicationContext applicationContext = new AnnotationConfigApplicationContext(SpringConfig.class);
UserInfo userInfo = applicationContext.getBean("userInfo", UserInfo.class);

@Service和@Controller内部都有一个@Component

# 2.4 BeanDefinition方式

AnnotationConfigApplicationContext applicationContext = new AnnotationConfigApplicationContext();
applicationContext.refresh();

AbstractBeanDefinition beanDefinition = BeanDefinitionBuilder.genericBeanDefinition().getBeanDefinition();
beanDefinition.setBeanClass(UserInfo.class);
applicationContext.registerBeanDefinition("userInfo", beanDefinition);

UserInfo userInfo = applicationContext.getBean("userInfo", UserInfo.class);

# 2.5 FactoryBean方式

@Component
public class SuFactoryBean implements FactoryBean {
    @Override
    public Object getObject() throws Exception {
        return new UserInfo();
    }

    @Override
    public Class<?> getObjectType() {
        return UserInfo.class;
    }
}


AnnotationConfigApplicationContext applicationContext = new AnnotationConfigApplicationContext();
applicationContext.refresh();

AbstractBeanDefinition beanDefinition = BeanDefinitionBuilder.genericBeanDefinition().getBeanDefinition();
beanDefinition.setBeanClass(SuFactoryBean.class);
applicationContext.registerBeanDefinition("suFactoryBean", beanDefinition);

# 2.6 Supplier方式

AnnotationConfigApplicationContext applicationContext = new AnnotationConfigApplicationContext();
applicationContext.refresh();

applicationContext.registerBean(UserInfo.class, new Supplier<UserInfo>() {
    @Override
    public UserInfo get() {
         UserInfo userInfo = new UserInfo();
         userInfo.setUserName("老苏");
         return userInfo;
    }
});

UserInfo userInfo = applicationContext.getBean("userInfo", UserInfo.class);
