# 第二章 是什么蒙蔽了爱？

# 1、异化的沟通方式

致力于满足某种愿望，却倾向于忽视人的感受和需要，以致彼此的疏远和伤害

# 2、道德批判

何为道德批判：

用道德标准来评判人，如果一个人的行为不符合我们的价值观，那他就被看作是不道德或邪恶的

属于道德批判：

“你的毛病是你太自私”，“这是不恰当的”。 

 如果同事非常在乎细节，他就是“有强迫症”；反之，如果我更在乎细节，他就是“粗心大意”

暴力的根源在于人们忽视批次的感受和需要

# 3、进行比较

比较也是一种批判

# 4、回避责任

淡化责任的例子：

不管你是否喜欢，有些事情你不得不做

忽视情感的例子：

你让我伤透了心

回避责任的例子：

1、为什么欺骗顾客：因为老板叫我这样做   
2、为什么我要打自己的小孩？因为他跑到街上去    
3、为什么喝酒？因为我是个酒鬼   
4、为什么打扫房间？因为我不得不做

回避责任的说法：

 我讨厌给学生评级。这样做不仅没用，而且学生会很紧张。但我不得不评级。因为这是学校政策

# 5、强人所难

 我们无法强迫他人按照我们的期待生活。   
 许多人相信，作为父母、老师我们的职责就是改变孩子，让他们循规蹈矩。
 我们可以提出各种要求，但是无法强迫孩子按照我们的期待生活。 
 盛气凌人并无济于事

# 6、小结

人天生热爱生命，乐于助人。可是，异化的沟通方式使我们难以体会到心中的爱。道德评判就是其中的一种，它将不符合我们价值观的人 看作是不道德的或邪恶的。进行比较也是一种评判，它会蒙蔽对人对己的爱意。异化的沟通方式还淡化了我们对自己的思想、情感和行为的 责任意识。此外，强人所难也会造成心灵的隔阂。
