# 第三章 区分观察和评论

非暴力沟通的第一个要素：观察

# 1、人类智力的最高形式

印度哲学家克里希那穆提曾经说：“不带评论的观察是人类智力的最高形式”

# 2、区分观察和评论

表达方式 观察和评论被混为一谈 区分观察和评论
使用的语言没有体现出评论的人对其评论负有责任 你太大方了 当我看到你把吃午饭的钱都给了别人，我认为你太大方了
把他人思想，情感或愿望的推测当做唯一的可能 她无法完成工作 我不认为她能完成工作。或她说：“我无法完成工作”
把预测当做事实 如果你饮食不均衡，你的健康就会出现问题。 如果你饮食不均衡，我就会担心你的健康会出问题
缺乏依据 老李花钱大手大脚。 老李上周买书花了一千元
使用形容词和副词时，把评论当做事实 老王长得很丑 老李对我没有吸引力

如果我们表达言过其实，别人就可能产生逆反心理，而不愿做出友善的回应。

# 3、小结

非暴力沟通的第一个要素是观察。将观察和评论混为一谈，别人就会倾向于听到批判，并反驳我们。非暴力沟通是动态的语言，不主张绝对化的结论。它提倡在特定的时间和情境中进行观察，并清除的描述观察的结果。例如，它会说“欧文在过去的5场比赛中没有进一个球”，而不是说“欧文是个差劲的前锋”。

# 4、练习

下面的句子，哪些只是描述观察结果而不含任何评论

1、”哥哥昨天无缘无故的对我发脾气“
2、“昨晚妹妹在看电视的时候啃指甲”
3、“开会时，经理没有问我的意见”
4、“我父亲是个好人”
5、“小李的工作时间太长了”
6、“小苏很霸道”
7、“小李经常不刷牙” 8、“姑姑在和我说话的时候爱发恼骚”

答案:
1、“无缘无故”是评论；我认为哥哥发脾气也是评论。 如果这样说：“哥哥告诉我，他生气了”这句话就是描述了观察结果而不带任何评论。
2、观察结果，无评论
3、观察结果，无评论
4、无观察结果，有评论。如果这样说：“在过去的25年中，父亲将他工资收入的十分之一捐给了慈善机构”这句话描述了观察结果而不带任何评论。
5、无观察结果，有评论。“太长了”是评论。如果这样说：“本周小李在办公室工作了60小时以上”这句话描述了观察结果而不带任何评论。
6、无观察结果，有评论。“霸道”是评论
7、无观察结果，有评论。“经常”是评论
8、无观察结果，有评论。“爱发恼骚”是评论
