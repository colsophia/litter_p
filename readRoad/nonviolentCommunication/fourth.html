# 第四章 体会和表达感受

非暴力沟通的第二个要素：感受

# 1、区分感受和想法

A.想法

我觉得我吉他弹得不好。

在这个句子中，我评估自己弹吉他好不好，而没有表达感受。

B.感受

作为吉他手，我有些失落。

如果认为自己吉他弹得不好，我可能会失落、郁闷或厌烦。

当我们说“我觉得”，我们常常并不是在表达感受，而是在表达想法

下面这些词也是在表达想法，而非感受。如：被抛弃，被羞辱，无人理睬，被利用，被打扰，被拒绝，被虐待，被贬低，无人赏识

# 2、建立表达感受的词汇

举例：

（1）用于表达我们的需要得到满足时的感受：

兴奋 喜悦 精力充沛 兴高采烈 感激 感动 乐观 自信 开心 振奋 平静 自在 舒适 放松 踏实 安全 温暖 放心 无忧无虑

（2）用于表达我们需要没有得到满足时的感受：

害怕 担心 焦虑 着急 紧张 悲伤 伤感 泄气 灰心 沮丧 愤怒 生气 不满 不耐烦 不开心 内疚 不舒服 昏昏欲睡 孤独 郁闷 失望

# 3、小结

非暴力沟通的第二个要素是感受。通过建立表达感受的词汇表，我们可以更清楚地表达感受，从而使沟通更为顺畅。在表达感受时，示弱有助于解决冲突。此外，非暴力沟通还对表达具有感受的词语与陈述想法、评论以及观点的词语作了区分。

# 4、练习

以下的句子是否表达了感受。
1、“我觉得你不爱我”
2、“你要离开，我很难过”
3、“如果你不和我打招呼，我会觉得你不在乎我”
4、“我觉得我被人误解了”
5、“你帮我的忙，我很开心”

答案:
1、没有表达出感受 2、表达了感受
3、“你不在乎我”是对他人态度的判断，不是感受
4、“被人误解”是对他人观点的判断
5、表达了感受
